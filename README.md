# 🍱 FeedForward — Food Donation App

> Connecting food donors with shelters, NGOs & families in need.

## 🚀 Features
- 🍱 Donate surplus food with pickup details
- 🔍 Browse & claim available donations
- 🙏 NGOs post food requests with urgency levels
- 📊 Dashboard to track donation impact

## 🛠️ Tech Stack
`HTML5` · `CSS3` · `Vanilla JavaScript` · No frameworks · No dependencies

## ▶️ Run Locally
```bash
git clone https://github.com/yourusername/feedforward.git
cd feedforward
open index.html
```
